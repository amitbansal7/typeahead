# typeahead

Suggestions for search engines.

## Technology Stack:

1. Spring boot
2. MongoDB
3. Data Structure : prefix Tree/ Trie
4. Bootstrap

## API


#### Insert (inserts word into the data structure)
POST - /word


#### Suggest (returns list of suggestions with sorted according to the frequency of the query)
GET - /word
